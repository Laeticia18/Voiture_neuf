# Car Dealership Website Project

This project is a car dealership website developed using HTML, CSS, and JavaScript. The composition of the project is as follows:

- HTML: 55.9%
- CSS: 32.9%
- JavaScript: 11.2%

## Features
- User-friendly interface for browsing available cars.
- Detailed information for each car.
- Contact form to reach out for inquiries.

## Installation
To install and run the project locally, follow these steps:
1. Clone the repository: `git clone https://github.com/Laeticia18/Voiti-n-f`
2. Open `index.html` in your browser.
